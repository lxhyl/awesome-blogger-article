# awesome weibo  
 *记录有趣的博主,文章*

灵感来源于[河森堡](https://weibo.com/u/5992829552)老师的[一条博文](https://weibo.com/5992829552/L67SQsvDQ).    
我觉得很有意思，但由于微博的实效性，即使关注了博主，也有可能错过有趣的东西，更何况茫茫人海中，遇到合胃口的博主更是难求。所以想着做个文档网站，来汇总，分类，持续集成这些博主，文章。


> 如果是手机或者小屏设备，你可以在左下角找到目录。
